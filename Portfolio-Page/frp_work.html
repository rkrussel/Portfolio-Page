Find-a-Servicer

This is a project built with ReactJS to search for service providers who repair fitness equipment. The table containing the search results is a reusable component which can also be seen on the "Thank You" page of the checkout process and in order confirmation emails.

The data is retrieved from a back-end API service which I also built utilizing PHP and Laravel. The request queries either a zip code or state database table, which is than cross referenced with the addresses of the service providers using an algorithm to calculate the distance.
